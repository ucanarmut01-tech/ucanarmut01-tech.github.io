Privacy Policy

This Privacy Policy explains how we handle user data in our Discord verification bot.

1. Data We Collect

  • Email address (for verification)
  • Discord User ID

2. How We Use Data

Data is used only for verification and access control purposes.

3. Data Sharing

We do not sell, share, or distribute user data to third parties.

4. Data Storage

User data is stored securely and used only within the verification system.

5. User Rights

Users can request deletion of their data by contacting server administrators.
